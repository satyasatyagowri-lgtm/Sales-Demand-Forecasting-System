# Sales & Demand Forecasting System

## Project Overview
This project is a Machine Learning based system that predicts future sales using historical data. It helps businesses plan inventory, manage stock, and improve decision making.

## Features
Data cleaning and preprocessing. Time-based feature creation. Sales forecasting using Machine Learning. Visualization of sales trends.

## Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib.

## Methodology
Historical sales data is cleaned and prepared. Time-based features such as date and month are created. A regression model is trained to predict future sales. Results are visualized for better understanding.

## Example Output
Predicted Sales: 5000 units  
Trend: Increasing demand

## Conclusion
This project demonstrates how Machine Learning can help businesses forecast demand and make better decisions.
